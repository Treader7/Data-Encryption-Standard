# Data Encryption Standard
Foundation of computer Science Coursework
## Project Overview
The project implements the Data encryption standard (DES) algorithm in Python. The implmentation includes the complete DES encryption/decryption process, key schedule generation, and permuation and S-boxes arrays.

## Project Structure
- des_algorithm.py: Main file for using des algorithm
- permutations.py: Contains all permutation arrays and key shift schedule
- sboxes.py: Containes all S-boxes and a lookup and substitution function
- key_schedule.py: Contains the key schedule for encryption and decryption
- modes.py: Modes of operation
- Requirements.txt: Dependancies
- .gitignore: Gitignore file
