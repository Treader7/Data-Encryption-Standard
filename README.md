# Data Encryption Standard
Foundation of computer Science Coursework
## Project Overview
The project implements the Data encryption standard (DES) algorithm in Python. The implmentation includes the complete DES encryption/decryption process, key schedule generation, and permuation and S-boxes arrays.

## Project Structure
- des_algorithm.py: Main DES encryption/decryption
- permutations.py: All permutation tables
- sboxes.py: S-boxes and substitution functions
- key_schedule.py: 16 round key generation
- modes.py: ECB and CBC modes of operation
- Requirements.txt: Python dependancies
- .gitignore: Git ignore file
- License: MIT opensoure license
- READ.md: This file
